description: Detailed task statistics — counts by type, priority, and label breakdown
---

Display detailed task statistics for the current brain project.

Use `tasks_list` with `status: open` to get all open tasks, then compute and present:
- Task counts by status (open, in_progress, blocked, done)
- Distribution by priority (P0-P4)
- Distribution by task type (task, bug, feature, epic, spike)
- Label breakdown via `tasks_labels_summary`

Present as a formatted dashboard. For a quick health check, use `/status` instead.
